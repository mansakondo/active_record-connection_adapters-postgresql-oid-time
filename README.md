## Installation
Add this line to your application's Gemfile:

```ruby
gem "active_record-connection_adapters-postgresql-oid-time",
  github: "mansakondo/active_record-connection_adapters-postgresql-oid-time",
  branch: "main"
```

And then execute:
```bash
$ bundle
```
